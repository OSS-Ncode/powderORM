title: CockroachDB

---

CockroachDB는 PostgreSQL 와이어 프로토콜을 그대로 쓰기 때문에, Powder에서는
**별도 백엔드 없이 PostgreSQL 드라이버로 바로** 붙습니다. 런타임 쿼리와
마이그레이션 전부 지원해요.

## 연결

```
postgres://user:pass@host:26257/dbname
```

`postgres://` URL을 그대로 쓰면 됩니다 (CockroachDB 기본 포트는 26257).

## 동작 방식

- **자리표시자는 그대로 `?`** — 엔진이 `$n`으로 자동 변환해요 (문자열
  리터럴·주석·`::` 캐스트는 보존).
- **DDL / 마이그레이션** — `powder ddl --dialect postgres`가 만드는
  표준 DDL(`BIGINT`/`DOUBLE PRECISION`/`TEXT`/`BOOLEAN`)을 CockroachDB가
  그대로 받아들여요. `powder migrate --db postgres://...`도 동일하게 동작.
- **결과 캐시 없음** — 서버 DB는 다른 클라이언트의 쓰기를 감지할 수 없어
  캐시를 끕니다 (정확성 우선).

## 알아둘 것

CockroachDB는 분산 DB라 PostgreSQL과 완전히 같지는 않아요 — 일부 PG 기능
(저장 프로시저 등)이 없거나 다르게 동작합니다. Powder가 생성하는 DDL과
ORM 쿼리는 호환 범위 안이지만, raw SQL로 PG 전용 기능을 쓸 때는
[CockroachDB 호환성 문서](https://www.cockroachlabs.com/docs/stable/postgresql-compatibility)를
확인하세요.
